Pool Table Time Management System With Locking Integration using Arduino (Capstone Project)

×

The project prototype features a self-built wooden pool table designed to simulate an automated lock system with time and pricing management. This table includes six pocket holes equipped with servo motors that automatically lock once a player's allotted time ends. The system integrates hardware and software components, where an Arduino microcontroller connects the physical table with a Windows Forms application. The application manages session time, calculates pricing, and sends signals to the Arduino to trigger servo motor actions. A PC or laptop runs the application, enabling real-time control and monitoring of the table’s operational status.

On the software side, the project uses a Windows Forms application developed in C#. Each pool table is represented through a DataGridView, showing remaining time and corresponding prices. Admin staff can add time, view history, and generate earnings reports through various built-in features. Data management is handled using Firebase, which stores session histories and earnings in real time. The system is designed around three key components: the software interface, the hardware (table with Arduino and servo motors), and the cloud-based database for history tracking and report generation.

Project Overview


In this project, our client owns a pool table shop that accommodates multiple players daily. To improve efficiency and streamline operations, we developed a system where the pocket holes of the pool table automatically lock using servo motors once a player's time has ended. This automation helps the client manage sessions more effectively and reduces the need for constant manual monitoring.
